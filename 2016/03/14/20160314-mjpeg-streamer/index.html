安装mjpg-streamer的步骤：

我这里是基于笔记本的系统测试的，如果需要移植到一些嵌入式开发板上，操作类似。
参考Ubuntu 12.10下源码安装MJPG-Streamer及使用

首先安装一些依赖的软件：

$ sudo apt-get install libjpeg-dev subversion imagemagick

mjpg-strreamer依赖于libjpeg提供的函数。
subversion就是常说的svn，安装svn的目的是后面需要用svn下载mjpg-strreamer的源码
imagemagick具体不知道是为了啥，网上说是一个免费的创建、编辑、合成图片的软件。它可以读取、转换、写入多种格式的图片。

其次下载源码，下载官方的源码方法如下：

$ mkdir download
$ cd download
$ svn co https://mjpg-streamer.svn.sourceforge.net/svnroot/mjpg-streamer mjpg-streamer

网上有一些从github上下载的地址，但那些都是派生的版本。注意sourceforge上有一个缺省的下载包mjpg-streamer-r63.tar.gz，但这个包直接编译总是不过，特别是最后链接时老是报undefined reference todlopen’`之类的错误，没功夫去解决，先放一边吧。

$ cd mjpg-streamer-code/mjpg-streamer
$ make clean all

测试的方法：

$ export LD_LIBRARY_PATH=.
$ ./mjpg_streamer -i "input_uvc.so" -o "output_http.so -w ./www"

想要查看视频，请在web页面中输入地址：http://127.0.0.1:8080

参数说明：

  • -i 指定输入插件内容
  • -o 制定输出插件内容
  • -y 表示数据为yuv格式
  • -d (制定视频输入设备)
  • -w 192.168.9.111:8080(网络视频输出使用ip 和端口号)

mjpg-streamer的工作流程，大致如下所示：

plugins\input_uvc

5.init_videoIn函数
init_videoIn函数本身不复杂，大部分都是对vdIn 结构变量vd初始化。
接着掉用init_v4l2函数：

6.init_v412函数
init_v412里面主要是使用ioctl设置驱动相关属性。
有篇文章写得很清晰：
http://www.cnblogs.com/emouse/archive/2013/03/04/2943243.html

8.input_run函数
定义在mjpg-streamer/plugins/input_uvc/input_uvc.c里面，这个函数比较简单，主要完成：
1.分配帧内存
2.创建线程cam，线程函数cam_thread主要完成视频抓取功能

9.cam_thread函数
定义在mjpg-streamer/plugins/input_uvc/input_uvc.c里面，主要完成：
1.调用uvcGrab函数抓取视频帧数据。
2.调用compress_yuyv_to_jpeg或memcpy_picture拷贝帧数据到pglobal->buf，这2个都是系统提供的函数。

10.uvcGrab函数
位置：在mjpg-streamer/plugins/input_uvc/input_uvc.c
功能：抓取视频帧

11.memcpy_picture函数
位置：V4L2 接口提供的函数
功能：拷贝数据到指定内存中

plugins/output_http

output_init函数
定义在mjpg-streamer/plugins/output_http/output_http.c里面，这个函数比较简单，主要完成：
1.解析–output参数项，这个和input_init函数几乎是一样的。
2.初始化一些结构，包括侦听端口等信息

12.output_run函数
位置：mjpg-streamer/plugins/output_http/output_http.c
功能：为客户端提供服务，传输视频数据

函数结构比较简单，创建线程，线程函数为server_thread

13.server_thread函数
位置：mjpg-streamer/plugins/output_http/httpd.c
功能：创建TCP socket，为客户端提供网络服务

14.client_thread函数
位置：mjpg-streamer/plugins/output_http/httpd.c
功能：为客户端发送数据

主要调用函数send_snapshot和send_stream发socket发送数据
